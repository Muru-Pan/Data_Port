# Muruga Pandian Portfolio

A professional portfolio website for a Data Scientist & Analyst.

## Tech Stack
- **React 19**
- **Vite**
- **Tailwind CSS v4**
- **Framer Motion**
- **Lucide React**

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## Contact Form Setup
The contact form is pre-configured to work with **EmailJS**. To enable it:
1. Sign up at [emailjs.com](https://www.emailjs.com/).
2. Create a Service, Template, and Public Key.
3. Update `src/components/Contact.jsx` with your credentials.
